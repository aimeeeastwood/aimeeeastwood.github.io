What is the difference between Margin, Border, and Padding?

Margin

Margins are external elements and exists "outside" the html file. Usually is determines the space between the physical edge of the screen and the edge of the html file. To think of it as the "outside space" is pretty accurate.

Border

The border is literally that. A border between the "outside space" and the "inside space". It is the picture frame in which the content sits. The border can be either visable or invisable but it is there regardless.

Padding

Padding is the amount of "inside space" between the border and the content. Setting the padding will determine where inside the border (or picture frame) the content will sit.
